Caminhos Mínimos

Algoritmo de Dijkstra

Método projetado para encontrar caminhos mínimos em grafos ponderados, onde cada aresta possui um custo associado.

  • Expande primeiro os ramos com menor custo acumulado a partir do vértice de origem.
  • Reaproveita subcaminhos já otimizados para construir soluções maiores com consistência.

Aplicação direta: encontrar as menores rotas de uma origem para todos os outros vértices de uma rede.
